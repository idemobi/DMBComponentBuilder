# DMBComponentBuilder Glossary

## Purpose

Define common terms used in `DMBComponentBuilder` documentation and AI instructions.

## Project-specific section

When copying this file to another PageBuilder ecosystem project, update this section first.

- Project name: `DMBComponentBuilder`
- Project folder: `DMBComponentBuilder`
- Project role: reusable visual component package for PageBuilder and BootstrapBuilder applications.
- Publication host: `labs_idemobi_com`

## Terms

### Component builder

A fluent type that renders one reusable visual component, such as a code block, file tree, roadmap, timeline, FAQ, rating badge, or social share block.

### Component definition

A data object that describes one item or entry rendered by a component, such as a step, timeline entry, roadmap item, file tree node, or social share option.

### Component state

An enum or value that controls visual state, such as active, completed, pending, warning, error, or disabled.

### Composer

A helper type that centralizes reusable class, effect, or markup composition for one component family.

### Embedded asset

A static asset included in the package and exposed to consuming MVC/Razor applications through component configuration.

### Example partial

A Razor partial in `labs_idemobi_com` that demonstrates one component state or usage scenario.

### DocumentationViewer

The documentation browsing feature in `labs_idemobi_com` that displays generated API documentation for NuGet packages.

### DocumentationBuilder

The documentation generation process that extracts and renders API documentation. AI prepares content; the developer executes the generator.
